# Hack Yeah 2024 Krakow

## Developer's set-up ✨

#### 🎯 Install dependencies

`flutter pub get`

#### 🧱 Activate and initialize mason

`dart pub global activate mason_cli`

`mason init`

#### 🍈 Activate and initialize melos

`dart pub global activate melos`

`melos bootstrap`

#### 🔨 Build the app

`melos run build`
